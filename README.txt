This plugin synchronises wordpress users with a mail-lite group subscribers.

== Description ==

This plugin will synchronise wordpress users with a specified mailer lite group. 

Upcoming features.

* Specific groups for different user roles
* Compatibility with other lead generation plugins 

== Installation ==

1. Upload `wp-mailer-lite.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
